⭕ 步骤 1：安装 Git
   操作：下载并安装 Git for Windows
   网址：https://git-scm.com/download/win
   验证：运行 git --version
   
⭕ 步骤 2：配置 Git
   命令：
   git config --global user.name "您的名字"
   git config --global user.email "your@email.com"
   
⭕ 步骤 3：创建 GitHub 账号
   网址：https://github.com
   操作：注册并验证邮箱
   
⭕ 步骤 4：初始化本地仓库
   命令：
   git init
   git add .
   git commit -m "Initial commit"
   
⭕ 步骤 5：推送到 GitHub
   操作：
   1. 在 GitHub 创建新仓库
   2. 关联并推送
   命令：
   git remote add origin https://github.com/用户名/geo-info-app.git
   git branch -M main
   git push -u origin main
   
⭕ 步骤 6：连接 Vercel
   网址：https://vercel.com
   操作：
   1. 用 GitHub 登录
   2. 导入 geo-info-app 项目
   3. 点击 Deploy
   
⭕ 步骤 7：测试网站
   操作：访问 Vercel 提供的网址
   测试所有功能是否正常
